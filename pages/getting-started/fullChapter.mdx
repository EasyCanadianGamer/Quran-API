---
title: How to get an Surah?
description: How to get an surah/chapter from the Quran using the Quran API.
---

## Request Format

Make a **GET** request to `/<surahNo>.json`

Replace `<surahNo>` with the Surah Number.

> **NOTE:** `surahNo` must be between `1` and `114`

## Response

A _JSON_ object containing the whole surah.

### Example Response
