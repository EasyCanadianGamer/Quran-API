---
title: How to get an Ayah?
description: How to get an ayah or verse from the Quran using the Quran API.
---

import { Callout } from "nextra/components";


## Request Format

Make a **GET** request to `/<surahNo>/<ayahNo>.json`

<Callout type="info">Replace `<reciterNo>` with the Reciter Number ( `1` or `2` )</Callout>

<Callout type="info">Replace `<surahNo>` with the Surah Number and `<ayahNo>` with the Ayah Number</Callout>

<Callout type="warning">`surahNo` must be between `1` and `114` <br/> `ayahNo` must be between `1` and the total number of Ayahs in the Surah</Callout>


## Response

A _JSON_ object containing informations about the Ayah.

### Example Response

Endpoint: [`/api/1/2.json`](https://quranapi.pages.dev/api/1/2.json)
