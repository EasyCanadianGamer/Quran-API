---
title: How to get an Ayah?
description: How to get an ayah/verse from the Quran using the Quran API.
---

## Request Format

Root URl : `quranaudio.pages.dev` ( for audio )

Make a **GET** request to `/<reciterNo>/<surahNo>_<ayahNo>.mp3`

> 1 : Mishary Rashid Al-Afasy

> 2 : Abu Bakr Al-Shatri

Replace `<reciterNo>` with the Reciter Number ( `1` or `2` )

Replace `<surahNo>` with the Surah Number and `<ayahNo>` with the Ayah Number.

> **NOTE:** `surahNo` must be between `1` and `114`. `ayahNo` must be between 1 and the total number of Ayahs in the Surah.

## Response

A mp3 file will be returned.

### Example

Endpoint: [`/2/1_2.json`](https://quranaudio.pages.dev/2/1_2.mp3)
