---
title: How to get an Ayah?
description: How to get an ayah/verse from the Quran using the Quran API.
---

## Request Format

Make a **GET** request to `/<surahNo>/<ayahNo>.json`

Replace `<surahNo>` with the Surah Number and `<ayahNo>` with the Ayah Number.

> **NOTE:** `surahNo` must be between `1` and `114`. `ayahNo` must be between 1 and the total number of Ayahs in the Surah.

## Response

A _JSON_ object containing informations about the Ayah.

### Example Response

Endpoint: [`/api/1/2.json`](https://quranapi.pages.dev/api/1/2.json)
