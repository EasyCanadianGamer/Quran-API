## Request Format

Make a **GET** request to [`/reciters.json`](https://quranapi.pages.dev/api/reciters.json)

## Response

A _*JSON*_ object containing the list of available reciters.

### Example Response

Endpoint: [`/reciters.json`](https://quranapi.pages.dev/api/reciters.json)

```json copy
{
    "1": "Mishary Rashid Al-Afasy",
    "2": "Abu Bakr Al-Shatri"
}
```
