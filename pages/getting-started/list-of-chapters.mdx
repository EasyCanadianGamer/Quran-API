---
title: Get all Surahs' info from the Quran
description: Get all Surahs' info from the Quran. This is a free API to get all Surahs' info from the Quran. It returns the Surahs' names, number of verses, and more.
---

## Request Format

Make a **GET** request to `/api/surah.json`

## Response

An array of all Surahs' info from the Quran.
